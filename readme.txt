This plugin will add a button that allows users to scroll smoothly to the top of the page.

== Description ==

This is small wordpress plugin for one click scroll to top. If you use this plugin you can go bottom to top your website one click. 


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `scroll-up-wordpress` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After complete activation process you will get a menu below settings named "Scroll To Top" in where change the option of this plugin.


== Frequently Asked Questions ==

1. How do I install this plugin?

You can install as others regular wordpress plugin. No different way. Please see on installation tab.
